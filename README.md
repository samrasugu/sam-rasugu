Personal portfolio built with [NextJS](https://nextjs.org), [TypeScript](https://www.typescriptlang.org/), and [TailwindCSS](https://tailwindcss.com/).

## How to run

First, install all required packages:
```bash
npm install
```

Then run the development server:
```bash
make dev
```
or
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
